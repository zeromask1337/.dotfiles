# Vue Expert References

This directory contains detailed Vue.js documentation references for advanced topics.

## Available References

These files can be created on-demand from the full Vue documentation at https://vuejs.org/llms-full.txt

Potential reference files:
- `accessibility.md` - WCAG guidelines, ARIA, semantic HTML, screen reader support
- `animations.md` - Transition components, CSS animations, JavaScript hooks
- `sfc-spec.md` - Complete SFC specification and compiler details
- `application-api.md` - Application-level configuration and global APIs
- `reactivity-api.md` - Deep dive into reactivity system internals
- `render-functions.md` - h(), render functions, JSX
- `typescript.md` - Advanced TypeScript patterns and typing
- `testing.md` - Component testing with Vitest and Vue Test Utils

## Usage

Reference files are loaded lazily when specific topics are needed. The main SKILL.md contains the essential Vue patterns for 90% of use cases.
